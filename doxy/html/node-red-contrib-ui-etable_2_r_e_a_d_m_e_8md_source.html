3 Notes:
4  .This node was previously called node-red-node-ui-etable, please
5  remove it before using this node
6  .The versioning will now follow the Tabulator Library versioning
7 
8 This node is based on the capabilities of the Tabulator library and allows
9 the user to define complex - editable tables and constructs.
10 
11 Credit must go to Kazuhito Yokoi for his inspiration. node-red-node-ui-table
12 
13 This node differs from node-red-node-ui-table due to it being more native to
14 the Tabulator libraries, it allows very complex editable tables to be built, although this means that the configuration of the node now must follow the http://tabulator.info json format.
